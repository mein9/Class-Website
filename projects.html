The To-Do List project is an interactive web application that allows users to manage their daily tasks efficiently. Users can add new tasks by typing them into the input field and clicking the "Add Task" button. Each task appears in a list with a "Remove" button next to it, allowing users to delete tasks once completed. The project showcases basic DOM manipulation and event handling in JavaScript.

Number Guessing Game

I have selected a random number between 1 and 100. See if you can guess it in 10 turns or fewer. I'll tell you if your guess was too high or too low.

The Number Guessing Game is a fun and interactive game where users try to guess a randomly selected number between 1 and 100. Users have up to 10 attempts to guess the correct number. After each guess, the game provides feedback on whether the guess was too high, too low, or correct. The project demonstrates the use of JavaScript for game logic, user input handling, and dynamic content updates.
